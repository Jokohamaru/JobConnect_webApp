Không cần đọc hết tất cả. Chọn file phù hợp với mục đích của bạn.

### 2. Sử dụng Table of Contents
Mỗi file có structure rõ ràng. Jump đến section bạn cần.

### 3. Bookmark các section quan trọng
- API endpoints
- Test accounts
- Troubleshooting
- Environment variables

### 4. Đọc code examples
Các file có nhiều code examples. Đọc và thử chạy.

### 5. Follow links
Các file có cross-references. Follow để hiểu sâu hơn.
